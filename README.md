# Music Generation with AI 🎵

## Project Overview

This project uses Artificial Intelligence and Deep Learning to generate music. An LSTM (Long Short-Term Memory) neural network is trained on MIDI note sequences and learns musical patterns. After training, the model generates new note sequences, which are converted into a MIDI file.

## Features

- Extracts notes from MIDI files
- Preprocesses musical note sequences
- Trains an LSTM neural network
- Generates new music notes
- Creates a MIDI output file

## Technologies Used

- Python
- TensorFlow / Keras
- Music21
- NumPy
- Google Colab

## Project Workflow

1. Load MIDI music data
2. Extract note sequences
3. Convert notes into numerical format
4. Train LSTM model on note patterns
5. Generate new note sequences
6. Convert generated notes into MIDI format
